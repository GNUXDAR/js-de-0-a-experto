v-on || @

Vue.js proporciona una directiva denominada v-on que puede enlazar a cualquier evento, como v-on:click. Como el control de eventos es una tarea esencial, Vue.js también proporciona el método abreviado de teclado @ para cualquier evento. Por tanto, para enlazar un evento de clic, puede usar el método abreviado de teclado @click.

Controladores de eventos

Puede crear controladores de eventos si agrega funciones al campo methods de una aplicación o un componente de Vue. El campo methods es como data(), pero en lugar de devolver objetos de estado, mantiene una lista de las funciones disponibles para la aplicación. Puede hacer referencia a estas funciones en el código HTML de la misma manera que hace referencia a otras funciones de JavaScript.

Nota La razón clave para agregar funciones al campo methods es que las funciones puedan acceder a los datos registrados.

Al agregar un método a una aplicación o un componente de Vue, this apunta a la instancia activa. Cualquier dato disponible para la instancia activa, como name en el ejemplo siguiente, es accesible desde this.
