Presentación de la directiva v-for

Para mostrar todos los elementos de una lista, puede usar la directiva v-for. v-for se comporta de forma similar a un bucle for...of de JavaScript. Itera por una colección y proporciona acceso a cada elemento por mesio de la variable que se declara.

La sintaxis v-for tiene este aspecto: 

		
			v-for="itemName incollectionName"

collectionName es el nombre de la matriz en el objeto de datos. En cada iteracción, itemName~ asume la identidad del elemento actual. Para usar v-for en una plantilla, agregue la directiva dentro de una declaración de elemento. Al hacerlo, se repite el elemento HTML tantas veces como elementos haya en la matriz.

Nota

Si va a crecar una lista sin ordenar (ul), o tendria que agregar v-for al elemento (ul). Agréguelo al elelemtno (li), ya que ele lemento (li) es lo que se repetirá.
