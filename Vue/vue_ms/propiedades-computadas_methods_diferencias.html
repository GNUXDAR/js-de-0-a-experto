En Vue puede usar propiedades computadas para descargar cálculos y otras formas de cadenas dinámicas.

Una propiedad computadas es una función que devuelve un valor. Al igual que un método, una propiedad computadas puede acceder a la instancia activa de Vue mediante this.

Puede usar las propiedades calculadas para combinar firstName y lastName en una propiedad fullName, realizar búsquedas en una matriz para devolver el valor correcto o realizar otras tareas dinámicas.

Diferencias

  • computed: Se utilizan para calcular valores derivados, basados en otros propiedades reactivas. Se cachean, esto las hace mas eficiente.
  • methods: Se utilizan para definir funciones que se ejecutan en respuesat a eventos o acciones del usaurio. Son utilespara logica que debe ejecutarse explicitamente, como al hacer click en un boton. No cacheada, se vuelven a ejecutar cada vez que se llaman.
